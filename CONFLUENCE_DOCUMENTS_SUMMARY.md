## üìã How to Use Confluence Documents

### Option 1: Direct Copy-Paste (Recommended)
1. Open the `*_CONFLUENCE.md` file
2. Copy the entire content or specific sections
3. In Confluence, click "Edit" on your page
4. Paste the content
5. Confluence will render the markup automatically

### Option 2: Section-by-Section
1. Open the Confluence-formatted document
2. Copy individual sections (h1, h2 blocks)
3. Paste into corresponding Confluence pages
4. Better for organizing content across multiple pages

### Option 3: Import via Confluence API
1. Use Confluence REST API to import content
2. Convert Markdown to Confluence Storage Format
3. Use automation tools for bulk import

## üé® Confluence Markup Features Used

All Confluence-formatted documents use:

- **Headings:** `h1.`, `h2.`, `h3.` for section headers
- **Panels:** `{panel:title=...}` for highlighted information boxes
- **Code Blocks:** `{code:language=...}` for code examples
- **Tables:** `||Header||` syntax for formatted tables
- **Checkboxes:** `( )` and `(/)` for task lists
- **Info/Warning Panels:** `{info:}`, `{warning:}` for alerts
- **Bold/Italic:** `*bold*`, `_italic_` for emphasis
